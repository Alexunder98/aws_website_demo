# AWS website app demo
A fun way to learn and show DevOps skills :sparkles: [here](www.alexct.website) :sparkles:

## Features

AWS Services:
AWS EC2, AWS Route 53, AWS SecurityGroups, AWS IAM

Networking:
DNS, Routing, SSH, Public IP, Private IP, NginX Reverse Proxy, Certificates

App:
Python, Flask, Systemd Service

## How it works

User connects to www.alexct.website then his request reaches global DNS server > AWS DNS Server > AWS EC2 instance > Nginx > APP
